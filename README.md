# Bienvenido a QuizzIT!!

Esta es una aplicación móvil de preguntas y respuestas sobre distintas áreas de la informática, creada con React Native utilizando el framework Expo y Supabase.

## Para empezar, sigue estos pasos:

1. Instalar dependencias

   ```bash
   npm install
   ```

2. Iniciar el servidor de desarrollo

   ```bash
   npx expo start
   ```
3. Escanear el código QR con tu dispositivo móvil para abrir la aplicación en Expo Go.
